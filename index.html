Case studies

App Automation — Login module

Goal: validate the end-to-end login flow on Android (happy path + negative cases). Design: BDD with Cucumber + Page Object Model. Common waits/actions live in BasePage, locators & actions in LoginPage, behavior in Gherkin (login.feature), and glue/reporting in LoginSteps.

What I implemented

  • Structured POM to isolate locators and actions from test logic.
  • Explicit waits centralized to reduce flakiness.
  • Readable scenarios in Gherkin (+ background to prep the app state).
  • Step logs for clear reporting; ready for CI and device/cloud runs.

Key outcomes

  • Faster maintenance: UI changes contained to one page object.
  • Traceable, human-readable test intent via BDD.
  • Stable runs thanks to consistent waits and reusable helpers.
BasePage.java (shared actions) 

// minimal excerpt
public class BasePage {
              
    protected AppiumDriver driver;
    protected WebDriverWait wait;
    
    public BasePage(AppiumDriver driver) {
        this.driver = driver;
        this.wait = new WebDriverWait(driver, 10);
    }
    protected void click(By locator) {
        wait.until(ExpectedConditions.elementToBeClickable(locator)).click();
    }
    protected void writeText(By locator, String text) {
        wait.until(ExpectedConditions.elementToBeClickable(locator)).sendKeys(text);
    }
LoginPage.java (locators + actions) 

public class LoginPage extends BasePage {

    private By saltarBtn = By.xpath("//android.view.ViewGroup//*[contains(@text,'Saltar')]");
    private By phoneField = By.xpath("//android.widget.EditText[@content-desc='Campo seguro Número AT&T (10 dígitos)']");
    private By passField = By.xpath("//android.widget.EditText[@content-desc='Campo seguro Ingresar tu contraseña']");
    private By ingresarBtn = By.xpath("//android.view.ViewGroup[@content-desc='Botón Ingresar']");

 public LoginPage(AppiumDriver driver) {
        super(driver);
    }

    public void clickSaltar() {
        try {
            if (isElementPresent(saltarBtn, 5)) {
                click(saltarBtn);
            }
        } catch (Exception e) {
            System.out.println("Botón 'Saltar' no presente. Continuando.");
        }
    }

LoginSteps.java (glue + logs) 

public class LoginSteps {

    AppiumDriver driver = DriverFactory.getDriver();
    LoginPage loginPage = new LoginPage(driver);

    @Given("que la app está abierta")
    public void que_la_app_esta_abierta() {
        ExtentsReportsDemo.test.log(Status.INFO, "App inicializada correctamente");
    }

    @When("doy clic en el botón Saltar")
    public void doy_clic_en_el_boton_saltar() {
        loginPage.clickSaltar();
        ExtentsReportsDemo.test.log(Status.PASS, "Botón 'Saltar' clickeado correctamente (si estaba presente)");
    }
